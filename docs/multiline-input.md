# 複数行入力ガイド

vibe-local-go は複数行のテキスト入力に対応しています。

## 入力方法

### 1. コピー＆ペースト（推奨）

複数行のテキストをそのままターミナルにペーストできます。ブラケットペーストモード対応により、改行を含むテキストが正しくバッファに挿入されます。Enter で送信。

### 2. Ctrl+J / Alt+Enter

入力中に `Ctrl+J` を押すと、その位置に改行が挿入されます。`Alt+Enter` も同様です。継続行には `... ` プロンプトが表示されます。Enter で全行を一括送信します。

```
> 1行目の入力
... 2行目（Ctrl+Jで改行）
... 3行目
↵ Enter で送信
```

複数行モード中の操作：

| キー | 動作 |
|------|------|
| `↑` / `↓` | 行間のカーソル移動 |
| `←` / `→` | 文字単位のカーソル移動 |
| `Ctrl+A` | 現在行の先頭へ |
| `Ctrl+E` | 現在行の末尾へ |
| `Ctrl+K` | 現在行のカーソル以降を削除 |
| `Ctrl+W` | 現在行内で単語削除 |
| `Ctrl+U` | 全行クリア |
| `Enter` | 送信 |

単一行の場合は `↑` / `↓` で従来通り入力履歴をナビゲートできます。

### 3. `"""` ブロックモード

`"""` を入力すると複数行モードに入り、再度 `"""` を入力すると確定します。

```
> """
  (""" で終了)
... 1行目
... 2行目
... 3行目
... """
```

### 4. バックスラッシュ `\` 継続

行末に `\` を付けると次の行に継続します。

```
> これは長い文章で\
... 次の行に続きます
```

## 備考

- **Shift+Enter** はターミナル環境では Enter と同じバイト（CR）を送信するため区別できません。代わりに `Ctrl+J` を使用してください。
- ペースト時は CR, LF, CRLF いずれの改行コードにも対応しています。
